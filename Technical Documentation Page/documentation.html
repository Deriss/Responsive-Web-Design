Introduction to Flexbox

The CSS flexbox allows the design of a flexible responsive layout structure in a simple way.

To start using flexbox, the first step is to define a flex container. This is the principal element that contains the different elements that will be rearranged responsively. To do this, simply set the property display:flex; in CSS for the container.

Flex-direction

The flex-direction property defines if the elements would be stack in a row or in a column. The elements are stacked in a row as default.

  • flex-direction:row;
  • flex-direction:column;

Flex-wrap

The flex-wrap property sets whether flex items are forced onto one line or can wrap onto multiple lines. If wrapping is allowed, it sets the direction that lines are stacked.

The following values are accepted:

  • flex-wrap:nowrap; the flex items are laid out in a single line which may cause the flex container to overflow.
  • flex-wrap:wrap; the flex items break into multiple lines.
  • flex-wrap:wrap-reverse; behaves the same as wrap but the elements are disposed in the opposite order.

Flex-flow

The flex-flow CSS property is a shorthand property for flex-direction and flex-wrap properties.

For example: flex-flow:row nowrap;

See flex-direction and flex-wrap for details on the values.

Justify Content

The justify-content property aligns the flexible container's items when the items do not use all available space on the main-axis (horizontally).

The values that this property can have are:

  • justify-content:flex-start; default value. Items are positioned at the beginning of the container.
  • justify-content:flex-end; items are positioned at the end of the container.
  • justify-content:center; items are positioned at the center of the container.
  • justify-content:space-around;items are positioned with space before, between, and after the lines.
  • justify-content:space-between; items are positioned with space between the lines.
  • justify-content:initial; sets this property to its default value.
  • justify-content:inherit; inherits this property from its parent element.
